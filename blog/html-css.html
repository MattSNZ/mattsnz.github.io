The Box Model as a House

A key part of CSS is using the box model. This uses the combination of content, padding, border, and margin to position HTML elements on the page.

It looks like this:

A good way to think of the box model is of your element as a house.

If the content of the element is the space inside of your house, then the border would be your exterior walls. Between the internal and external walls, is the padding. Hopefully in your case thereâ€™s insulation in there amidst all the newspapers and coffee cups that the builders left behind.

Wrapping up this terrible metaphor is the margin, which would then be the space between the external walls of your house and the boundary of your property.

Helpful or Interesting Things to Look Out For:

One thing that you need to look out for when spacing your elements is margin collapse. While this sounds a bit terrifying, in reality it means that, in some situations, only the larger of the top or bottom margins between elements will be respected.

These situations are:

  • Adjacent siblings
  • No content separating parents or children
  • Empty blocks

You can find out more at the MDN Web Docs. For now, here's a code example:

HTML

<p>Lorem ipsum</p>

<p>Lorem ipsum dolor sit</p>

CSS

p {
margin: 10 20 15;
}

You might expect a gap of 25px between those two <p> elements but, when the page loads, what will happen is that the margin-bottom of the top element will collapse and you'll be left with 15px between them.

One last helpful tip is that you can use element { margin: auto; } with any block element and it will center centre it automatically. So handy!
